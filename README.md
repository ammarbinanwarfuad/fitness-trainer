# Fitness Trainer Website

A modern, responsive fitness trainer website built with HTML, CSS, and JavaScript. This website provides information about fitness services, workout programs, and includes a functional BMI (Body Mass Index) calculator.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **BMI Calculator**: Interactive BMI calculator with additional metrics including:
  - BMI calculation and categorization
  - BMR (Basal Metabolic Rate) calculation
  - TDEE (Total Daily Energy Expenditure) calculation
- **Team Section**: Showcase of fitness trainers
- **Transformation Stories**: Inspirational success stories from members
- **Modern UI**: Beautiful dark theme with gradient effects and smooth transitions

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid)
- JavaScript (Vanilla JS)
- Font Awesome Icons
- Google Fonts (Inter, Miama)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/fitness-trainer.git
   ```

2. Navigate to the project directory:
   ```bash
   cd fitness-trainer
   ```

3. Open `index.html` in your web browser

## Usage

### BMI Calculator

1. Enter your height in centimeters (cm)
2. Enter your weight in kilograms (kg)
3. Enter your age
4. Select your sex (Male/Female)
5. Select your activity level:
   - Sedentary (little or no exercise)
   - Lightly active (light exercise 1-3 days/week)
   - Moderately active (moderate exercise 3-5 days/week)
   - Very active (hard exercise 6-7 days/week)
   - Extra active (very hard exercise, physical job)
6. Click "Calculate" to see your BMI, BMR, and TDEE

## BMI Categories

- **Underweight**: BMI below 18.5
- **Healthy**: BMI between 18.5 and 24.9
- **Overweight**: BMI between 25.0 and 29.9
- **Obese**: BMI 30.0 and above
